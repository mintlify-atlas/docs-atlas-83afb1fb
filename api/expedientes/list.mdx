## Descripción

Este endpoint devuelve una lista completa de todos los expedientes registrados en el sistema, incluyendo todos los campos de información de las etapas inicial, seguimiento y final.

## Endpoint

```
GET /api/Expedientes/Lista
```

  -H "Content-Type: application/json"

## Códigos de Estado

<ResponseField name="200" type="OK">
  La solicitud se procesó correctamente. Revise el campo `Exito` para determinar si hubo errores en el procesamiento.
</ResponseField>

## Manejo de Errores

En caso de error, la respuesta incluirá:

```json
{
  "Exito": false,
  "Mensaje": "Descripción del error ocurrido",
  "Data": null
}
```
